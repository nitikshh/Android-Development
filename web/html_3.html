Width and Height in CSS

Certainly! The `width` and `height` properties in CSS are used to specify the width and height of an element, respectively. Here's an example code that demonstrates the usage of `width` and `height` properties: 

<!DOCTYPE html>
<html>
<head>
  <title>Width and Height Example</title>
  <style>
    /* CSS Styles */
    .box {
      width: 200px;
      height: 150px;
      background-color: #f2f2f2;
      border: 1px solid #ccc;
      padding: 10px;
      margin-bottom: 20px;
    }
    
    .full-width {
      width: 100%;
    }
    
    .half-height {
      height: 50%;
    }
  </style>
</head>
<body>
  <div class="box">Fixed Width and Height</div>
  
  <div class="box full-width">Full Width</div>
  
  <div class="box half-height">Half Height</div>
</body>
</html>

Explanation : In the above code, we have a `.box` class that represents a box-shaped element. It has a fixed `width` of 200 pixels and a fixed `height` of 150 pixels. It also has some padding, border, and background color applied to it.

The second box has an additional class of `full-width`, which sets the `width` property to `100%`, making it expand to the full width of its container.

The third box has an additional class of `half-height`, which sets the `height` property to `50%`, making it occupy half of the height of its container.

You can modify the values of `width` and `height` to suit your requirements and apply them to different elements as needed.

Box Sizing in CSS

The `box-sizing` property in CSS allows you to control how the width and height of an element are calculated, taking into account the element's padding and border. There are two commonly used values for `box-sizing`:

1. `content-box` (default): This is the default behavior where the width and height of an element only include the content and exclude the padding and border. This means that if you specify a width of 300px for an element with 10px padding and 2px border, the total width of the element will be 324px (300px + 10px left padding + 10px right padding + 1px left border + 1px right border).

2. `border-box`: With this value, the width and height of an element include the content, padding, and border. This means that if you specify a width of 300px for an element with 10px padding and 2px border, the total width of the element will be 300px, and the padding and border will be included within that width.

Here's an example code that demonstrates the use of `box-sizing` property: 

<!DOCTYPE html>
<html>
  <head>
    <title>Box Sizing Example</title>
    <style>
      /* Apply box-sizing: border-box to all elements */
      * {
        box-sizing: border-box;
      }

      /* Style for the container div */
      #container {
        width: 300px;
        padding: 20px;
        border: 2px solid black;
      }

      /* Style for the inner div */
      #inner {
        width: 100%;
        height: 100px;
        padding: 10px;
        border: 1px solid red;
      }
    </style>
  </head>
  <body>
    <div id="container">
      <div id="inner">Content</div>
    </div>
  </body>
</html>

Explanation : In the above code, we have a container `<div>` with an ID of `container` and an inner `<div>` with an ID of `inner`. We apply the `box-sizing: border-box;` property to all elements using the universal selector (`*`) in the CSS code. This ensures that the `border-box` box-sizing model is applied to all elements on the page.

The container `<div>` has a fixed width of 300px, 20px padding, and a 2px solid black border. The inner `<div>` has a width of 100% (relative to the container), a fixed height of 100px, 10px padding, and a 1px solid red border.

Since we've set `box-sizing: border-box;`, the width of the container `<div>` remains at 300px, including the padding and border. Similarly, the inner `<div>` width is calculated based on the container's width, and the padding and border are included within that width.

This ensures that the total width of the inner `<div>` remains within the boundaries of the container, taking into account the padding and border, without causing any unexpected overflow.

Font Family in css

In CSS, the `font-family` property is used to specify the font family or typeface for text elements. You can define multiple font families in a comma-separated list, allowing the browser to apply the first available font from the list.

Here's an example code that demonstrates the use of `font-family` property: 

<!DOCTYPE html>
<html>
  <head>
    <title>Font Family Example</title>
    <style>
      /* Apply font family to the body */
      body {
        font-family: Arial, sans-serif;
      }

      /* Apply different font families to specific elements */
      h1 {
        font-family: "Times New Roman", Times, serif;
      }

      p {
        font-family: Verdana, Arial, sans-serif;
      }
    </style>
  </head>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Explanation : In the above code, we set the `font-family` property for the `body` element to `Arial, sans-serif`. This means that the browser will attempt to use the Arial font first, and if it's not available, it will fallback to a generic sans-serif font.

For the `h1` element, we specify a different font family using `font-family: "Times New Roman", Times, serif;`. Here, we provide a specific font name `"Times New Roman"` and also include fallback options of `Times` and a generic serif font.

Similarly, for the `p` element, we define the `font-family` property as `Verdana, Arial, sans-serif;`. This means that the browser will attempt to use Verdana font first, then Arial, and if neither is available, it will fallback to a generic sans-serif font.

To add custom fonts from Google Fonts, you need to follow these steps:

1. Visit the [Google Fonts website](https://fonts.google.com/) and choose the desired font(s) you want to use.

2. Click on the "+" icon next to the font(s) you selected to add them to your collection.

3. Once you've added all the desired fonts, click on the "Family Selected" bar that appears at the bottom of the page.

4. In the popup that appears, you'll find the CSS code snippet to include in your HTML file. Copy that code.

5. Paste the CSS code inside the `<head>` section of your HTML file.

6. In your CSS code, you can now reference the selected Google Font using the `font-family` property.

Here's an example that uses a Google Font: 

<!DOCTYPE html>
<html>
  <head>
    <title>Google Fonts Example</title>
    <link
      href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap"
      rel="stylesheet"
    />
    <style>
      /* Apply Google Font to specific elements */
      h1 {
        font-family: "Roboto", sans-serif;
        font-weight: 700;
      }

      p {
        font-family: "Roboto", sans-serif;
        font-weight: 400;
      }
    </style>
  </head>
  <body>
    <h1>This is a heading with a Google Font</h1>
    <p>This is a paragraph with a Google Font.</p>
  </body>
</html>

In the above code, we include the Google Font "Roboto" by adding the `<link>` tag to the Google Fonts CSS file in the `<head>` section of our HTML. This makes the font available for use in our CSS code.

We then apply the font to the ` h1` and `p` elements using `font-family: "Roboto", sans-serif;`. The `font-weight` property is also set to control the font's weight (700 for bold, 400 for regular).

Text Align in CSS

In CSS, the `text-align` property is used to specify the horizontal alignment of text within an element. It controls the positioning of the text relative to the element's content box.

Here are the commonly used values for the `text-align` property:

1. `text-align: left;` - Aligns the text to the left side of the content box.
2. `text-align: center;` - Centers the text horizontally within the content box.
3. `text-align: right;` - Aligns the text to the right side of the content box.
4. `text-align: justify;` - Adjusts the spacing between words in a line to justify both the left and right edges of the content box.
5. `text-align: start;` - Aligns the text according to the browser's default language-specific direction. (For left-to-right languages, it behaves the same as `left`, and for right-to-left languages, it behaves the same as `right`).
6. `text-align: end;` - Aligns the text opposite to the browser's default language-specific direction. (For left-to-right languages, it behaves the same as `right`, and for right-to-left languages, it behaves the same as `left`).

Here's an example code that demonstrates the use of `text-align` property with different values: 

<!DOCTYPE html>
<html>
  <head>
    <title>Text Align Example</title>
    <style>
      .container {
        width: 300px;
        border: 1px solid black;
        padding: 10px;
      }

      /* Apply different text alignments */
      .left-align {
        text-align: left;
      }

      .center-align {
        text-align: center;
      }

      .right-align {
        text-align: right;
      }

      .justify-align {
        text-align: justify;
      }
    </style>
  </head>
  <body>
    <div class="container left-align">
      <p>This text is left-aligned.</p>
    </div>

    <div class="container center-align">
      <p>This text is center-aligned.</p>
    </div>

    <div class="container right-align">
      <p>This text is right-aligned.</p>
    </div>

    <div class="container justify-align">
      <p>
        This text is justified. It will have extra spacing between words to
        align both the left and right edges of the content box.
      </p>
    </div>
  </body>
</html>

Explanation : In the above code, we create a `.container` class to define a container element with a fixed width, border, and padding. We then create different classes (`.left-align`, `.center-align`, `.right-align`, and `.justify-align`) to apply different text alignments to the paragraphs within the containers.

By adding the respective alignment class to each container, we can control the text alignment within the paragraphs.

Flex Container

In CSS, the `display: flex` property is used to create a flex container, which enables a flexible layout for its child elements. Within a flex container, you can use various properties to align and position the child elements.

Here are some commonly used properties for centering the flex container and its contents:

1. `display: flex;` - Creates a flex container.
2. `align-items: center;` - Aligns the flex items vertically along the cross-axis. In this case, it centers the items vertically within the flex container.
3. `justify-content: center;` - Aligns the flex items horizontally along the main axis. In this case, it centers the items horizontally within the flex container.

Here's an example code that demonstrates the use of `display: flex`, `align-items`, and `justify-content` properties to center the flex container and its contents: 

<!DOCTYPE html>
<html>
  <head>
    <title>Flexbox Center Example</title>
    <style>
      .container {
        display: flex;
        align-items: center;
        justify-content: center;
        height: 300px;
        border: 1px solid black;
      }
      
      .box {
        width: 200px;
        height: 100px;
        background-color: lightblue;
      }
    </style>
  </head>
  <body>
    <div class="container">
      <div class="box"></div>
    </div>
  </body>
</html>

Explanation : In the above code, we create a flex container with the class `.container`. The `display: flex` property is applied to the container to enable flexbox layout. We also set `align-items: center` and `justify-content: center` to center the flex items (in this case, a single `.box` element) both vertically and horizontally within the container.

By adjusting the `height`, `width`, and other properties, you can customize the size and appearance of the flex container and its contents.

Additionally, there are other properties you can use to control the flex layout, such as:

- `flex-direction` - Specifies the direction of the flex container's main axis (row or column).
- `flex-wrap` - Determines whether flex items should wrap to a new line when there is not enough space.
- `align-self` - Overrides the `align-items` property for an individual flex item.
- `order` - Controls the order in which flex items are displayed.
- `flex-grow`, `flex-shrink`, `flex-basis` - Determines how flex items should grow, shrink, and be sized.

These properties provide further flexibility and control over the layout of the flex container and its child elements.

Shadow in CSS

In CSS, the `box-shadow` property is used to add shadow effects to elements. It allows you to create visually appealing shadows around elements, providing depth and dimension to your design. The `box-shadow` property has the following syntax: 

box-shadow: [horizontal offset] [vertical offset] [blur radius] [spread radius] [color];

Here's an explanation of each property:

1. `Horizontal Offset`: Specifies the horizontal distance of the shadow from the element. A positive value pushes the shadow to the right, while a negative value pushes it to the left.

2. `Vertical Offset`: Specifies the vertical distance of the shadow from the element. A positive value pushes the shadow downwards, while a negative value pushes it upwards.

3. `Blur Radius`: Determines the blur effect of the shadow. A higher value creates a more blurred and spread-out shadow, while a lower value creates a sharper shadow. A value of `0` means no blur.

4. `Spread Radius`: Controls the spread of the shadow. A positive value expands the size of the shadow, while a negative value shrinks it. A value of `0` means the shadow is the same size as the element.

5. `Color`: Specifies the color of the shadow. This can be any valid CSS color value, such as a named color, hex code, RGB value, or HSL value.

Here's an example that demonstrates the usage of the `box-shadow` property with different properties: 

<!DOCTYPE html>
<html>
  <head>
    <title>Box Shadow Example</title>
    <style>
      .box {
        width: 200px;
        height: 200px;
        background-color: lightgray;
        box-shadow: 5px 5px 10px 0px rgba(0, 0, 0, 0.5);
      }
      
      .box2 {
        width: 200px;
        height: 200px;
        background-color: lightgray;
        box-shadow: -5px -5px 10px 0px rgba(0, 0, 0, 0.5);
      }
      
      .box3 {
        width: 200px;
        height: 200px;
        background-color: lightgray;
        box-shadow: 0px 0px 10px 5px rgba(0, 0, 0, 0.5);
      }
    </style>
  </head>
  <body>
    <div class="box"></div>
    <div class="box2"></div>
    <div class="box3"></div>
  </body>
</html>

Explanation : In the above code, we have three `<div>` elements with different box shadow effects applied. The `box-shadow` property is used to define the shadow for each element. The first `.box` has a shadow with a horizontal offset of `5px`, vertical offset of `5px`, blur radius of `10px`, and no spread radius. The second `.box2` has a shadow with negative horizontal and vertical offsets, creating an inward shadow effect. The third `.box3` has a shadow with no horizontal or vertical offsets, a blur radius of `10px`, and a spread radius of `5px`, resulting in a larger shadow around the element.

By adjusting the values of the `box-shadow` property, you can create different shadow effects to suit your design needs.

`*` symbol in CSS

The `*` symbol in CSS is used as a universal selector, which selects all elements on the page. When used in conjunction with other selectors, it allows you to apply a set of styles to all elements.

The `*` selector is often used to apply a global CSS reset or to set default styles for all elements. It can be useful when you want to override or normalize the default styles provided by browsers.

Here's an example code that demonstrates the use of the `*` selector with a set of CSS rules: 

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Explanation : In the above code, the `*` selector is used to target all elements on the page. The following CSS properties are applied to all elements:

1. `margin: 0;`: This sets the margin of all elements to `0`, removing any default margin.

2. `padding: 0;`: This sets the padding of all elements to `0`, removing any default padding.

3. `box-sizing: border-box;`: This sets the box-sizing property of all elements to `border-box`. This ensures that the width and height of elements include any padding and border, making it easier to work with layouts.

By using the `*` selector with these CSS rules, you can establish a consistent starting point for your styles across all elements on the page. It helps to create a more predictable layout and avoids unexpected spacing or sizing issues caused by default browser styles.

It's important to note that using the `*` selector with extensive styles can have performance implications, as it applies styles to every element. It's generally recommended to use it sparingly and only when necessary.
